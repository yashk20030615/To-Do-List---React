# React To-Do List

A simple and interactive To-Do List application built with React.js.  
This project helps you manage your daily tasks efficiently by adding, marking complete, and deleting tasks.

## Features

- Add new tasks with a simple input form  
- Mark tasks as completed/uncompleted  
- Delete tasks you no longer need  
- Responsive design, works on desktop and mobile  
- Clean and minimal UI using React functional components and hooks

## Technologies Used

- React.js (with functional components and hooks)  
- CSS for styling  
- JavaScript (ES6+)
